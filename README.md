# Direct Call | Voice Private Line

Une application de communication vocale minimaliste et ultra-performante, conçue pour contourner les restrictions et offrir une connexion directe et sécurisée entre deux personnes.

## Fonctionnalités
- **P2P Direct**: Communication de pair à pair sans serveur central (WebRTC).
- **Audio Haute Qualité**: Optimisé pour la voix avec une consommation de bande passante minimale.
- **Design Minimaliste**: Une interface sombre, propre et sans distractions.
- **PWA Ready**: Installez-la sur votre écran d'accueil comme une application native.

## Comment l'utiliser
1. Ouvrez l'application sur les deux téléphones/ordinateurs.
2. Copiez votre ID et envoyez-le à votre partenaire (ou vice versa).
3. Entrez l'ID de votre partenaire dans le champ prévu à cet effet.
4. Appuyez sur le bouton central pour appeler.

## Sécurité
Cette application utilise le protocole WebRTC avec un cryptage de bout en bout pour garantir que vos conversations restent privées.
